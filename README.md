# README

This data science project is built in the Course Data Science Lab at
Constructor University Bremen.

It investigates the relationship between Schwartz’s basic human values and political attitudes in Europe between 2018 and 2024. Using data from the European Social Survey (ESS), the project analyzes how values such as Universalism, Security, and Tradition predict political orientation, trust, and policy preferences across different generations and countries, with a special focus on pre- and post-pandemic changes.

The project is structured as a Quarto website and contains separate pages for each research question, including literature review, methodological explanations, and empirical analyses.

To explore the project, please use the navigation menu on the left side of the website. Each section corresponds to a specific research question and analysis.
